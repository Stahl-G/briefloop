# MABW-080 Public Experiment Fixtures

This directory contains public-safe experiment fixture material for the MABW-080
approved-guidance manifestation harness.

improves output quality. A complete controlled experiment still requires
operators to scaffold condition workspaces, run each condition, register runs,
build scorecards, import human assessment, and summarize the case.
