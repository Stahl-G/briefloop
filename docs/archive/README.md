# Archived Documentation

Documents in this folder are historical records: superseded architecture
references, dated memos, and one-off design notes. They are kept for lineage
and audit value only.

Rules:

- Archived documents are not updated for current behavior.
- Current docs, README surfaces, and skills must not cite archived documents as
  implementation or support truth. Use `docs/architecture-status.md` and
  `docs/support-matrix.md` instead.
- Archived cross-links may point at other archived documents.

Contents:

- `mabw-architecture-reference-v0.1.1.md` … `v0.3.0-legacy.md`,
  `v0.3.0-revision-roadmap.md` — superseded architecture references; the
  current reference is `docs/briefloop-architecture-reference-v0.3.0.md`.
- `architecture-memo-*-2026-06-11.md`, `design-note-preference-taste-governance-2026-06-11.md`
  — dated design memos from the 2026-06-11 architecture pass.
- `brief-loop-engineering.md`, `harness_matrix.md` — early engineering/harness
  design notes superseded by the charter, orchestrator contracts, and gates.
- `agent-collaboration.md`, `init-wizard-design.md`, `github-project.md`,
  `code-review-fix-guide.md`, `mas-v2-evaluation.zh-CN.md` — one-off design or
  evaluation notes whose outcomes are merged into current docs and code.
