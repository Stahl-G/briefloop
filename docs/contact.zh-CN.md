# 联系方式

BriefLoop 公开项目网站：[briefloop.ai](https://briefloop.ai)

可以通过以下公开邮箱联系：

| 渠道 | 邮箱 | 用途 |
|---|---|---|
| 问候 / 轻量反馈 | [hello@briefloop.ai](mailto:hello@briefloop.ai) | 初次联系、产品问题、轻量反馈 |
| 联系 / 合作 | [contact@briefloop.ai](mailto:contact@briefloop.ai) | 合作、试点、商务联系、贡献者协调 |
| 帮助 | [help@briefloop.ai](mailto:help@briefloop.ai) | 使用问题、安装问题、文档反馈 |
| 安全 | [security@briefloop.ai](mailto:security@briefloop.ai) | 私密安全问题或漏洞报告 |

提交安全问题时，请尽量包含受影响版本、复现步骤，以及能说明问题的最小安全示例。除非确有必要且已获授权，不要通过邮件发送密钥、客户数据、私有报告或未公开公司材料。

这些邮箱是人工联系渠道，不代表 SLA、交付审批、发布授权、法律建议、合规建议、投资建议，也不代表 BriefLoop 已经提供语义证明。
