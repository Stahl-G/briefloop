# v0.7.2 公开光伏集成运行摘要

这是一份 public-safe 的 reference summary。它记录一次基于公开光伏政策与市场材料的双轮 MABW 集成运行，用来说明控制面、改进记忆和门禁是否按设计工作。

这不是原始 trace zip，也不是 A-grade 因果实验。原始运行包保留在私有环境中，不公开发布，因为其中包含本地路径、运行 transcript 和调试痕迹。本文只公开不可逆的摘要、哈希前缀、行为结论和限制。

## 结论

定级：**B+ integration reference**

这次运行证明：

- MABW 可以在真实 runtime 中完成双轮简报流程。
- baseline run 没有 Improvement Memory materialized。
- 第二轮 run 将人工批准的 `AG-0001` guidance 冻结进 `improvement_memory_snapshot.md`。
- runtime manifest 记录了 `materialized_entry_ids: ["AG-0001"]`。
- quality gates 真实阻断过运行，并在修复后放行。
- workflow state、artifact registry、event log、quality gate report 等控制记录完整存在。

这次运行不证明：

- 输出质量提升。
- guidance 的稳定跨模型表现。
- 严格因果归因，因为 run1/run2 的 claim layer 哈希不同。
- A-grade controlled reference。A-grade 需要冻结同一套 candidate / screened / claim ledger，只改变 Improvement Memory 是否存在。

## Claim Layer 哈希

这次运行不能作为严格因果实验，原因是 run1/run2 的事实层文件不同：

| artifact | run1 sha256 前缀 | run2 sha256 前缀 |
|---|---|---|
| `candidate_claims.json` | `b623dc598956f74e` | `3e056076a5377339` |
| `screened_candidates.json` | `34102eddcfb010f9` | `2979c14f96f9c69d` |
| `claim_ledger.json` | `cd7c4edc57213b9f` | `d67016d14195bdea` |

因此，正确表述是：**观察到 approved guidance materialization 后的行为变化**，不是证明 guidance 是唯一因果变量。

## Reader Output 观察

运行中观察到的正向行为：

- run2 更频繁地先说明业务含义，再给事实背景。
- run2 对不确定性和风险边界的提示更明确。
- run2 的结构更接近管理层简报，而不是单纯新闻列表。

运行中暴露的问题：

- 当时的 Flash 输出曾出现 reader-facing 内部 claim ID 和流程词残留。
- 这直接推动了 v0.7.2 的 reader-final gate 和 delivery hardening。
- 因此，本文不发布原始 reader brief 作为 showcase output。

## Public-Safety 处理

公开内容只保留：

- 控制面结论；
- hash 前缀；
- materialization 证据摘要；
- 不含本地路径的运行统计；
- 不含公司私有事实的泛化 guidance 描述。

不公开：

- 原始 transcript；
- 本地绝对路径；
- 操作者私有路径或用户名；
- wrong-model raw pack；
- 含 reader-output residue 的原始 brief。

## 正确引用方式

可以说：

> v0.7.2 之前的一次 public-safe solar integration run demonstrated Improvement Memory materialization, active gate enforcement, and complete control records in a real runtime run.

不要说：

> MABW proved output quality improvement.

不要说：

> MABW proved approved guidance has a causal effect on the brief.

更准确的边界是：

> This is an integration reference, not a controlled causal evaluation. A-grade manifestation experiments are deferred to the v0.8 evaluation track.
